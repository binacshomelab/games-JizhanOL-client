## 1. 生成文件字符格式为 latin1 ==> 需修改行末字符

`æ&^@^@` 需替换为 `c^@^@`

>   注意: ^@ 其实是一个字符

**必须使用 vim 进入 visual 模式，并修改单字符**

1.   `v` 进入 visual 模式
2.   `y` 复制末尾字符
3.   `p` 粘贴字符
4.   替换 `æ&` 为 `c` 【正常单字符 c 】
